# An example todo app using Cordova, Lokijs, loki-cordova-fs-adapter.
---
To run it run:
```
$ npm i -g cordova
$ npm i --save-dev
$ cordova platform add <platform>
$ cordova run <platform>
```

To make changes:
```
1. make your changes
$ webpack
$ cordova run <platform>
```
